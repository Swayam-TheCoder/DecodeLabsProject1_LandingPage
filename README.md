# 🎓 TechStuds - Responsive Landing Page

A modern, responsive landing page for **TechStuds**, an online learning platform designed to help students master Full Stack Development through project-based learning.

This project was built using **HTML5, CSS3, and JavaScript** with a strong focus on responsive design, clean UI, and user experience.

---

## 🚀 Live Demo

```
https://decode-labs-project1-landing-page.vercel.app/
```

---

## ✨ Features

* Responsive Design (Mobile, Tablet & Desktop)
* Modern Glassmorphism Navigation Bar
* Smooth Scrolling Navigation
* Animated Mobile Sidebar Menu
* Hero Section
* Trusted Companies Section
* Featured Courses
* Learning Journey Timeline
* Why Choose Us Section
* Student Testimonials
* Call-to-Action Section
* Contact Form
* Professional Footer
* Hover Animations
* Clean & Semantic HTML5 Structure

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/TechStuds-Landing-Page.git
```

Open the project folder:

```bash
cd TechStuds-Landing-Page
```

Run the project by opening:

```
index.html
```

or use **Live Server** in Visual Studio Code.
